# Contributing to Swapy Campus

Welcome! Please follow these guidelines:

## Monorepo Layout
- `apps/backend`: Express & Prisma application.
- `apps/frontend`: Vite & React SPA.
- `apps/ai-service`: FastAPI Python service.

## Development Commands
- Start dev mode: `pnpm dev`
- Run tests: `pnpm test`
- Generate Prisma Client: `pnpm prisma:generate`
- Migrate Database: `pnpm prisma:migrate`
